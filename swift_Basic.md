# Basic

- Basic
  - 대소문자를 구분
  ```
  var a = 10
  var A = 20 // a 와 A 는 다른 변수임
  ```
  
  - 구문 끝의 세미콜론은 생략
  ```
  print(Hello)
  ```
  
  - 문자열, 문자에 큰 따옴표 사용
  ~~~
  "abc"
  "b"  // 문자열인지 문자인지 타입 명시를 꼭 해줘야 함 
  ~~~

  - 주석
  ~~~
  /* 여러줄 */
  var a = 10 // 한줄
  ~~~

- 변수와 상수
  - 변수 : 실행에 따라 변하는 값을 저장
  - 상수 : 변하지 않을 값을 저장
  ~~~
  var a // 변수
  let b // 상수
  ~~~
  - 한번 정해진 타입은 바꿀 수 없음
